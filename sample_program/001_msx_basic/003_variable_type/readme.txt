MSX-BASIC は、整数変数・単精度BCD浮動小数点(単精度実数)変数・
倍精度BCD浮動小数点(倍精度実数)変数・文字列変数
が扱えます。
変数名の後に、%,!,#,$ を記述することで、その型を指定できます。

	A% : 整数変数A
	A! : 単精度実数変数A
	A# : 倍精度実数変数A
	A$ : 文字列変数A

同じ名前でも、異なる型であれば、別の変数扱いです。

100 A%=100
110 A!=1.2345678!
120 A#=2.3456789012345#
130 A$="STRING"
140 PRINT A%
150 PRINT A!
160 PRINT A#
170 PRINT A$
180 PRINT A

型記号を省略すると # になります。

100 A%=100
110 A!=1.2345678!
130 A$="STRING"
140 PRINT A%
150 PRINT A!
170 PRINT A$
180 PRINT A

A# が無い状態で、A を参照しても、A# を作って初期値 0 を返します。

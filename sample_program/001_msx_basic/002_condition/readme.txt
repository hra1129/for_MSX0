BASICでは、条件演算子の評価結果は整数値になります。
条件が成立した場合は -1、不成立の場合は 0 と評価されます。

100 DEFINT A-Z
110 A=10
120 PRINT "A=10:"; (A=10)
130 PRINT "A=11:"; (A=11)
140 PRINT "A<10:"; (A<10)
150 PRINT "A<9 :"; (A<9)
160 PRINT "A>10:"; (A>10)
170 PRINT "A>11:"; (A>11)
180 PRINT "A<=10:"; (A<=10)
190 PRINT "A<=9 :"; (A<=9)
200 PRINT "A>=10:"; (A>=10)
210 PRINT "A>=11:"; (A>=11)

ちなみに、わかりやすさのために、条件演算子の前後に () を
付けていますが、これは無くても構いません。

A=B=10

この記述で、B=10 の比較結果である 0 か -1 が A に代入されます。

A= の = は代入命令、B= の = は条件演算子(比較演算子) ということですね。
記述が同じなので混同しがちですが、意味が違いますので区別して下さい。

漢字BASIC が搭載されている MSX では、CALL KANJI を実行することで、
漢字モードに切り替えられます。（元に戻す場合は CALL ANK）

■漢字モードの特徴
　CALL KANJI は、CALL KANJI0, CALL KANJI1, CALL KANJI2, CALL KANJI3
の4モードある。
　特に、CALL KANJI0 (0は省略可能) は、文字の大きさが大きくなるので
　MSX0 の小さな画面でも見やすい（はず）。
　LOCATE 0,0:PRINT "漢字" のようなことが出来るので、便利です。

■漢字モードの操作
　[CTRL]+[SPACE] を押すと、漢字入力モードと英数入力モードを切り替え
　られます。
　漢字入力モードにしたとき、MSX-JE が有れば連文節変換による入力、
　無ければ単漢字変換による入力になります。

　　連文節変換：日本語文章を入力して単語単位で確定していく普通の日本語入力
　　単漢字変換：漢字1文字毎に読みを入力して変換していく簡易入力

■ASCIIセーブ
　MSX-BASIC のプログラムは、ASCIIセーブとバイナリーセーブの2種類があります。

　　SAVE "ファイル名", A

　後ろの A を付けるとASCIIセーブになります。
　ASCIIセーブしたファイルは、普通の Shift-JIS なテキストファイル
　（グラフィック文字だけちょっと異なる）なので、漢字の文字は PC等で入力
　して、それを SDカード経由で MSX0へ持ってくるのが楽です。

　　LOAD "ファイル名"

　LOAD命令は、ASCIIセーブされたファイルなのか､バイナリーセーブされた
　ファイルなのか自動的に検出してくれるので、保存の時気をつければ問題ありません。

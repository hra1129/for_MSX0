画像などを使いたい場合はグラフィック画面を使いたくなる。
ここに、文字を書くには OPEN命令を使うことになる。

100 DEFINTA-Z:COLOR15,0,0:SCREEN5
110 OPEN"GRP:"FOROUTPUTAS#1
120 PSET(100,100):COLOR6,12:PRINT#1,"HELLO WORLD!!"
130 PSET(100,108):COLOR7,13:PRINT#1,"HELLO WORLD!!"
140 PSET(100,116):COLOR8,10:PRINT#1,"HELLO WORLD!!"
150 PSET(100,124):COLOR9,4:PRINT#1,"HELLO WORLD!!"
160 A$=INPUT$(1)

行が開始位置は、PSET(x,y)で、描画する文字列の左上座標を指定すれば良い。
色は、COLOR前景色,背景色 で指定すれば良い。

MSX-BASICの変数名は、最大2文字までである。
厄介なのは、「3文字目を指定してもエラーにならないこと」です。

下記のプログラムを実行してみて下さい。

	100 DEFINTA-Z
	110 PARAM1=10
	120 PARAM2=20
	130 PRINT "PARAM1=";PARAM1
	140 PRINT "PARAM2=";PARAM2

行110で、変数PARAM1 に 10 を代入していますが、
行130の print で出すと 20 になってしまいます。

PARAM1 は、変数PARAM1 ではなく、変数PA の扱いになっているため、
行120で、PARAM2 に格納すると、変数PARAM2 も、変数PA であるために、
20 が上書きされます。

そのため、上記プログラムは、
	PARAM1=20
	PARAM2=20
という実行結果になってしまうわけです。
PRINT "PA=";PA
を実行してみると、20 と表示されるのがわかると思います。

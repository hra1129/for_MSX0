漢字BASICでは、SCREEN5等のグラフィック画面でも LOCATE, PRINT で文字を描画できます。
OPEN"GRP:"は必要ありません。大きな文字が出るので、MSX0stackの液晶でも見やすい。

100 DEFINTA-Z:_KANJI:COLOR15,0,0:SCREEN5
110 FORI=0TO100:LINE(RND(1)*255,RND(1)*211)-(RND(1)*255,RND(1)*211),(IAND7)+2:NEXT
120 LOCATE0,0:PRINT"漢字BASICはグラフィック画面でもOK"
130 LOCATE0,10:COLOR2,4:PRINT"色も変えられます"
140 A$=INPUT$(1)
